# SmartPot minimalistické dizajny

V tomto ZIPe sú 4 vizuálne návrhy ako PNG a zároveň sú zapracované priamo do `frontend/src/components/Dashboard.jsx` ako prepínateľné varianty:

1. `Premium iOS` — odporúčaný finálny dizajn.
2. `Clean Plant` — svetlý kartový dashboard.
3. `Dark Minimal` — tmavý IoT štýl.
4. `Ultra Minimal` — veľmi čistý textový prehľad.

V appke otvor dashboard a hore prepni dizajn cez prepínač. Výber sa uloží do `localStorage`.
